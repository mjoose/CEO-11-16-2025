# Images Directory

## Logo File

The logo file used across all pages is: `images/image0.png`

The logo should be:
- Format: PNG (with transparency preferred)
- The logo includes both the circular emblem and "ADPAR CONSTRUCTION" text
- The logo will be automatically resized to 60px height in the navigation bar
- Maximum width: 300px (to ensure full visibility from left to right)
- Minimum width: 200px (to maintain readability)

## Logo Source

The logo is referenced from: `images/image0.png`

If you need to update the logo, replace the file at this location with your new logo file, keeping the same filename `image0.png`.
